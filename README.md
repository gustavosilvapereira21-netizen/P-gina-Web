Robloxia History

Este projeto é uma página web simples e inicial dedicada a documentar a história, os fatos e os principais acontecimentos ocorridos no universo do Roblox entre os anos de 2006 e 2026. O site funciona como um blog ou portal de registro mantido pelo autor Gusta.

O que a Página Faz
A página apresenta uma introdução amigável e limpa para os visitantes, destacando o tema principal do site, que é a evolução e os eventos marcantes de Robloxia nas últimas duas décadas. Ela serve como o ponto de partida ou a página inicial para posts futuros sobre o assunto.

Tecnologias Utilizadas
O desenvolvimento foi feito utilizando padrões básicos e nativos da web:

HTML5: Utilizado para criar a estrutura e a semântica da página, incluindo cabeçalho, corpo principal, parágrafos, títulos e a inserção da imagem do logotipo.

CSS3: Incorporado diretamente no arquivo para estilizar o visual do site, definindo uma cor de fundo vermelha escura para os blocos, centralizando os textos na tela, ajustando o alinhamento e controlando o tamanho da imagem exibida.

Estrutura Visual e Layout
O design do site conta com um cabeçalho de destaque onde é exibida uma mensagem de boas-vindas. Logo abaixo, a seção principal centraliza o logotipo oficial do projeto (que ilustra a transição entre o antigo e o novo logo do Roblox) e apresenta a primeira publicação do autor, identificando o criador e o objetivo do conteúdo.

Como Executar o Projeto Localmente
Por se tratar de um site estático e construído apenas com tecnologias que rodavam diretamente no navegador, o processo para visualizar a página é direto:

Salve o código em um arquivo chamado index.html no seu computador.

Certifique-se de ter a imagem com o nome correto na mesma pasta para que ela apareça na tela.

Dê um duplo clique no arquivo index.html para abri-lo e visualizá-lo em qualquer navegador de sua preferência.
